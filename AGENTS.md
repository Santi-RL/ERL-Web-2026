# ERL-Web-2026 - Guia rapida para agentes

## Resumen del proyecto
Sitio web estatico para Estudio Contable Rojas Lem, construido con Vite + Tailwind.
El objetivo inmediato es publicar un MVP funcional con contenido real, canales de
contacto operativos y paginas legales completas.

## Archivos de instrucciones (MD)

- `CHECKLIST-LANZAMIENTO-CONTENIDO.md`
  Lista de tareas no tecnicas para salida a produccion: contenidos reales,
  formularios, agenda, testimonios, logos, recursos, legales, etc.
  Es el checklist de "listo para publicar".

- `PRIORIDADES.md`
  Roadmap tecnico priorizado: performance, SEO tecnico, accesibilidad, seguridad,
  pipeline de imagenes y calidad. Se usa para decisiones tecnicas y deuda tecnica.

- `road-to-mvp.md`
  Plan de negocio/operativo hacia el MVP. Define alcance, dependencias, riesgos y
  pasos de publicacion. Es la hoja de ruta para coordinar stakeholders.

## Nota
Mantener estos documentos actualizados y sin contenido duplicado: lo no tecnico en
`CHECKLIST-LANZAMIENTO-CONTENIDO.md`, lo tecnico en `PRIORIDADES.md`, y la hoja de
ruta del MVP en `road-to-mvp.md`.
